# Project Overview

This project performs a comprehensive statistical business analysis on a retail sales dataset using Python. 
It applies core statistical techniques and visualizations to extract meaningful business insights that support data-driven decision making.

# Objectives

Analyze retail sales data using statistics
Understand data distribution and variability
Perform hypothesis testing
Calculate confidence intervals
Visualize insights clearly for business interpretation
